# Upgraded RPG Character Manager
***

![The main menu of the program.](mainmenusimplegradebook.JPG)
***
This is a program that lets the user add, remove, view, edit, and compare RPG characters.

## How to Use
***
1. Open the main.py file
2. Press the run button
3. Select one of the available options:
  - Create a character
  - Edit a character
  - View a character
  - Remove a character
  - Compare two characters
  - View all characterse
  - Roll dice

## List of Key Features
- It lets the user add characters
- It has a name, race, role, and backstory generator if the user doesn't want to fill them out (all at the same time)
- It lets the user edit a character (their name or backstory, leveling up the character, resetting skill points, or editing the inventory)
- It lets the user remove a character
- It lets the user view a character's information and see their stats in either a radar or bar graph
- It lets the user compare two characters and see which one has a higher stat in a certain area, and can even compare one stat on a graph.
- It lets the user edit a character's information, level them up, reset skill points, and edit their inventory.
- It lets the user see all their characters
- It lets the user roll a d4, d6, d8, d10, d12, d20, and a percentile roll.

## Installation instructions
This program needs the libraries NumPy, Faker, MatplotLib, and Pandas, as well as access to the Programming drive.
To download NumPy: pip install numpy
To download Faker: pip install faker
To download MatplotLib: pip install matplotlib
To download Pandas: pip install pandas
Type in these commands into the terminal or command prompt in order to download.

Here are documentations for each of these libraries in case this installation doesn't work:
- NumPy: https://numpy.org/
- Pandas: https://pandas.pydata.org/
- MatplotLib: https://matplotlib.org/
- Faker: https://faker.readthedocs.io/en/master/
On each of these websites, go to either Get Started or Installation Instructions or something similar, and it will help you install them.
